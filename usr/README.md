# WhatsApp Clone

A simple Flutter implementation of the WhatsApp mobile UI.

## Features
- Classic WhatsApp green theme
- Tabbed layout (Chats, Status, Calls)
- Dummy chats list

## Setup
1. Ensure you have Flutter installed.
2. Run `flutter pub get`.
3. Run `flutter run`.
