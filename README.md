# YouTube Channel Promo App

A Flutter application designed to promote a YouTube channel, featuring a beautiful UI to showcase videos, playlists, statistics, and channel information.

## Features
*   **Creator Dashboard:** Display channel banners, avatars, and subscriber counts.
*   **Video Feed:** Browse the latest video uploads.
*   **Playlists:** Horizontal scrollable playlists for categorized content.
*   **About Section:** Links, channel descriptions, and total statistics.
*   **Responsive:** Works beautifully on mobile, desktop, and web views.

## Tech Stack
*   Flutter
*   Dart
*   Material Design 3

## Setup
To run this project:
1. Ensure you have Flutter installed.
2. Clone the repository.
3. Run `flutter pub get`.
4. Run `flutter run`.
